Данный проект является учебным на курсе базового фронтенда у hits, ТГУ. 

Сам проект представляет из себя клиентскую часть для сайта с блогами.
В возможностях юзера зарегистрироваться и залогиниться, просматривать свой профиль, возможность посты разных сообществ и без
вступать в эти самые сообщества, а также комментировать и лайкать любые понравившиеся ему посты.

Вся серверная часть приложения была сконцентрирована в API: https://blog.kreosoft.space/swagger/index.html, предоставленной для работы.
